elegram Bot (Node.js)

A simple Telegram bot built using Node.js to explore backend fundamentals, API handling, and event-driven programming.

 Features
Responds to /start command with a welcome message
Handles basic text like “hi”
Demonstrates a binary search logic via a command
/remind command replies after a 2-second delay (async behavior)
Reacts to stickers by sending a ❤️ emoji
Simple message-based interaction system

What I Learned
How Telegram Bot API works
Event-driven programming in Node.js
Handling asynchronous functions (delays, responses)
Structuring backend logic for real-time interactions
Building logic-based command systems

 Tech Stack
Node.js
Telegram Bot API
JavaScript

This project was built as a learning exercise to understand backend development concepts through a real-world interactive system (Telegram bot).
